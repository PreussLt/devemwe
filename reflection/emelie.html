Midjourney 1 Midjourney 2
Mein Lieblingsthema war tatsächlich das Implementieren von Schriftarten über Google Fonts. Es hat mir Spaß gemacht, damit herumzuexperimentieren, und ich fand es überraschend, wie viel Charakter eine Seite durch eine (oder mehrere) Schriftarten bekommen kann. Dazu sucht man sich einfach eine Schriftart auf Google Fonts aus, kopiert den angegebenen Code und fügt ihn in sein eigenes HTML- und CSS-Dokument ein. Das Ganze ist auf den Bildern veranschaulicht.

Vorkentnise

In der Oberstufe hatte ich zwei Jahre lang das Wahlfach Informatik. Damals haben wir uns mit Python, JavaScript und allgemein dem Thema „Netzwerke“ auseinandergesetzt. HTML hatte ich ebenfalls einmal in der 6. Klasse im Informatikunterricht behandelt. Abgesehen von einem SAP „Snap“-Kurs (eine Art Bausatz-Coding) lässt sich das Lernen damals jedoch als frustrierende Erfahrung bezeichnen.

CSS

Ebenfalls mit etwas Frust, aber deutlich weniger Hilflosigkeit im Lernprozess als damals, kann ich nun folgenden Fortschritt vorweisen: Ein grobes Verständnis von Webseiten-Developer-Tools sowie deren Nutzung. Das Implementieren von Schriftarten in meinen HTML-Code. Viele weitere neue Erkenntnisse in HTML und – vollkommen neu für mich – das Benutzen von CSS. Hierbei vor allem das Coden von Grid- und Flexlayouts. Hilfreich waren besonders die Website https://www.w3schools.com/, Erklärungen von KI-Programmen wie beispielsweise You, einzelne YouTube-Videos und meine Freunde. Was ich von dem Gelernten allerdings direkt wieder vergessen habe, ist, wie man innerhalb der cmd.exe Ordner erstellt oder Verzeichnisse ändert

Feedback

Eine Sache, die ich sehr hilfreich gefunden hätte, wäre eine Präsentation gewesen, in der eine Sammlung von verschiedenen Dingen gezeigt wird, die man auf Webseiten mit HTML und CSS realisieren kann. Beispielsweise die Änderung einer Farbe beim Überfahren eines Feldes mit der Maus, Bilder oder Text, der sich aktiv auf der Webseite bewegt, Filter, die auf Bilder angewendet werden können, und vieles mehr. Dabei geht es mir weniger um die Erklärung, wie man das Ganze genau codet und umsetzt, sondern einfach darum, die Möglichkeiten aufzuzeigen. So hat man mehr Ideen im Kopf und einen größeren Überblick. Abegesehen davon, habe ich nichts in dem Modul vermisst.
